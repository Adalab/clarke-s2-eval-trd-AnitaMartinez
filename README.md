# Adivina el juego
## Evaluación Sprint 2 Adalab


Este juego consiste en generar un número al azar entre 1 y 100, y el jugador tiene que adivinarlo. El juego le da pistas de si el número que prueba es demasiado alto o bajo, y va contabilizando el número de intentos. Hasta que al final el jugador acierta el número.
